# 使用方法

## 訓練模型
```bash
echo "from 臺灣言語服務.模型訓練 import 模型訓練; 訓練=模型訓練(); 訓練.走()" | python manage.py shell  
```

## 執行服務
### 試驗用
```bash
python manage.py runserver
```

### 正式上機
建議使用gunicorn配合nginx
```bash
pip install gunicorn
gunicorn --timeout 100000 專案名.wsgi
```
