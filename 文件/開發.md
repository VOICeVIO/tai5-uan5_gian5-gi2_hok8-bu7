## 試驗
### 全部走一擺
```bash
echo 'drop schema public cascade;create schema public;' | psql $USER
gzip -c -d 臺灣言語資料庫20150903.sql.gz | psql $USER
echo "from 臺灣言語服務.模型訓練 import 模型訓練; 訓練=模型訓練(); 訓練.走()" | python manage.py shell
```

### 個別試驗
```bash
python -m unittest # 會訓練模型
```
